E-Learning Platform with Gamification - Styled + Error Handling
---------------------------------------------------------------

This package contains a full-stack scaffold (backend + frontend) with improved frontend styling and robust error handling.
It includes the same server and data seed from the previous detailed scaffold plus cleaned, styled React components.

Quick setup (macOS):
  1) Unzip and open project folder
  2) Backend:
     cd backend
     npm install
     cp .env.example .env
     npm run seed   # optional: adds sample users, course, quiz
     npm run dev
  3) Frontend:
     cd frontend
     npm install
     npm start
  4) Visit http://localhost:3000
